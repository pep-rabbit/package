# user-cli

Термінальна оболонка для пошуку аптек/точок через API проєкту DeLiki.

Це не основний спосіб роботи з API, а допоміжна заглушка для перевірки логіки та швидкого ручного пошуку. Основний користувацький інтерфейс планується як веб-версія, а `user-cli` потрібен лише як MVP-демонстрація сценарію.

## Що робить

- надсилає запити до API за містом і назвою медичної програми/препарату;
- показує знайдені організації в терміналі;
- виводить помилки та порожній результат у зрозумілому вигляді;
- працює як мінімальний тестовий клієнт без повноцінної бізнес-логіки.

## Статус

Проєкт перебуває у стані MVP.

Швидше за все, я не буду активно далі розвивати цю частину, бо термінальний варіант незручний для нормальної роботи з API та програє веб-версії за UX і практичністю.

Код написаний не найкращим способом, і це свідомо тимчасове рішення для перевірки базового сценарію.

## Запуск

1. Встановити залежності.
2. За потреби вказати адресу API через змінну `API_BASE_URL`.
3. Запустити `main.py`.

Приклад:

```bash
API_BASE_URL=http://127.0.0.1:8080 python main.py
```

## Примітка

Якщо потрібен нормальний користувацький сценарій, цей інтерфейс слід розглядати лише як тимчасове рішення. Основний фокус має бути на веб-версії.
