# ДеЛіки

## Пошук безоплатних ліків поруч

Пацієнт отримав е-рецепт за програмою «Доступні ліки», але не знає, яка аптека поруч реально відпускає препарат.

## Датасети data.gov.ua

1. Аптечні заклади за програмою «Доступні ліки» (НСЗУ):
	[https://data.gov.ua/dataset/3503ea5a-456d-4780-905b-b74e7d8f09cf](https://data.gov.ua/dataset/3503ea5a-456d-4780-905b-b74e7d8f09cf)
2. Оплати аптекам за реімбурсацією (НСЗУ):
	[https://data.gov.ua/dataset/959dca0a-9b74-41ff-a7c8-f8de6398a219](https://data.gov.ua/dataset/959dca0a-9b74-41ff-a7c8-f8de6398a219)

Оновлення: дані НСЗУ оновлюються регулярно протягом 2025 року.

## Функціонал MVP

- Користувач вводить місто і назву препарату (МНН).
- Система знаходить усі аптеки з договором НСЗУ.
- Ранжує аптеки за обсягом оплат (більше = активніше відпускає).
- Показує адресу, контакт, тип видачі.
- Зберігає «ТОП-5 аптек» у файл.
