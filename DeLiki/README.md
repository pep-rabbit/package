# ДеЛіки

## Пошук безоплатних ліків поруч

«ДеЛіки» — частина навчального проєкту, яка допомагає пацієнту з е-рецептом за програмою «Доступні ліки» швидко знайти аптеку, що працює з НСЗУ та потенційно реально відпускає потрібний препарат.

Сервіс приймає місто і назву препарату за МНН, знаходить аптеки з договором НСЗУ, відбирає ті, що працюють у вказаному населеному пункті, і ранжує їх за сумою виплат за реімбурсацією. Чим вищий `activity_score`, тим активніше аптека виглядає в цій вибірці.

## Джерела даних

1. Аптечні заклади за програмою «Доступні ліки» (НСЗУ): [data.gov.ua](https://data.gov.ua/dataset/3503ea5a-456d-4780-905b-b74e7d8f09cf)
2. Оплати аптекам за реімбурсацією (НСЗУ): [data.gov.ua](https://data.gov.ua/dataset/959dca0a-9b74-41ff-a7c8-f8de6398a219)

Дані НСЗУ оновлюються регулярно протягом 2025 року. У цьому модулі вони збережені локально в папці `src/data` і використовуються для пошуку без звернення до зовнішнього API під час запиту.

## Як це працює

1. `src/core/logic.py` читає два CSV-файли через `polars`.
2. Довідник аптек фільтрується за містом.
3. Дані оплат групуються за `legal_entity_edrpou` і перетворюються в рейтинг `activity_score`.
4. Обидва набори даних з’єднуються по коду юридичної особи.
5. Аптеки відбираються за наявністю потрібного медичного програмного напряму в полі `medical_programs_in_divisions`.
6. Результат сортується за `activity_score` у спадному порядку і обрізається до топ-5.

## API

### `GET /api/top-pharmacies`

Параметри запиту:

- `city` — назва міста.
- `medical_program` — назва МНН або медичної програми для пошуку.

Повертає список аптек з такими полями:

- `legal_entity_name`
- `division_name`
- `division_addresses`
- `division_phone`
- `division_type`
- `division_settlement`
- `activity_score`

Приклад запиту:

```text
/api/top-pharmacies?city=Київ&medical_program=Інсуліни
```

## Технології

- `litestar` для HTTP API
- `polars` для швидкої обробки CSV та агрегацій
- `granian` як ASGI-сервер
- `msgspec` для серіалізації відповіді

## Структура модуля

- `src/main.py` — точка входу застосунку.
- `src/endpoin.py` — API-ендпоінт `/api/top-pharmacies`.
- `src/core/logic.py` — логіка пошуку і ранжування аптек.
- `src/data/` — локальні датасети НСЗУ.

## Запуск

Після встановлення залежностей сервіс стартує через `granian`:

```bash
granian src.main:app --host 0.0.0.0 --port 8080 --interface asgi
```
